# Pomodoro Timer

A simple timer app for use with [Pomodoro Technique](https://francescocirillo.com/pages/pomodoro-technique).

## Functionalities
- 25-minute timer (one pomodoro)
- 5-minute timer (short break)
- 20-minute timer (long break)
